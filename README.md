# Latviešu treneris

Тренажёр латышского: глагол *būt* по временам, склонения существительных и личный словарь (латышский → русский). Работает офлайн, ставится как приложение на iPhone и на Windows.

---

## Что внутри папки

```
index.html            ← само приложение
manifest.json         ← описание приложения (имя, иконка)
service-worker.js     ← делает работу офлайн + установку
icons/                ← иконки приложения
README.md             ← этот файл
```

Все эти файлы нужно залить на GitHub в один репозиторий.

---

## Шаг 1. Создай репозиторий на GitHub

1. Зайди на https://github.com → нажми зелёную кнопку **New** (или **+** → *New repository*).
2. Имя репозитория: `latviesu-treneris` (можно любое, латиницей, без пробелов).
3. Поставь галочку **Public** (для бесплатного GitHub Pages репозиторий должен быть публичным).
4. Нажми **Create repository**.

## Шаг 2. Залей файлы (через сайт, без командной строки)

1. На странице нового репозитория нажми ссылку **uploading an existing file** (или вкладка **Add file → Upload files**).
2. Открой папку с этими файлами на компьютере, выдели **всё** (index.html, manifest.json, service-worker.js, README.md и **папку icons целиком**) и перетащи в окно браузера.
   - Важно: папка `icons` должна загрузиться вместе с картинками внутри. Если перетаскивание папки не сработало — загрузи сначала корневые файлы, потом отдельно зайди в *Add file → Upload files* и перетащи содержимое папки icons, указав путь `icons/`.
3. Внизу нажми зелёную кнопку **Commit changes**.

## Шаг 3. Включи GitHub Pages

1. В репозитории открой **Settings** (шестерёнка сверху).
2. Слева выбери **Pages**.
3. В разделе *Build and deployment* → *Source* выбери **Deploy from a branch**.
4. В *Branch* выбери **main** и папку **/ (root)**, нажми **Save**.
5. Подожди 1–2 минуты, обнови страницу. Вверху появится ссылка вида:

   ```
   https://ТВОЙ-ЛОГИН.github.io/latviesu-treneris/
   ```

   Это адрес твоего приложения. Открой его — должно загрузиться.

---

## Шаг 4. Установка на iPhone

1. Открой ссылку **именно в Safari** (не в Chrome — на iPhone установка работает только из Safari).
2. Нажми кнопку **Поделиться** (квадрат со стрелкой вверх, внизу экрана).
3. Прокрути вниз и выбери **На экран «Домой»** (*Add to Home Screen*).
4. Нажми **Добавить**.

Теперь на домашнем экране появится иконка «Latviešu». Открывается как обычное приложение, на весь экран, работает без интернета.

## Шаг 5. Установка на Windows

1. Открой ссылку в **Chrome** или **Microsoft Edge**.
2. Справа в адресной строке появится значок установки (монитор со стрелкой) — нажми его.
   - Или: меню браузера (⋮) → **Установить приложение / Install app**.
3. Подтверди **Установить**.

Появится отдельное окно и иконка в меню «Пуск». Можно закрепить на панели задач.

---

## Как обновлять приложение потом

Если захочешь поменять что-то в приложении:

1. Открой нужный файл в репозитории на GitHub → карандаш (**Edit**) → внеси правки → **Commit changes**.
2. Чтобы устройства гарантированно подхватили новую версию, открой `service-worker.js` и поменяй номер версии в строке:
   ```js
   const CACHE = 'lv-treneris-v1';   // → поменяй на v2, v3 и т.д.
   ```
3. На телефоне/ПК просто переоткрой приложение — оно само обновится.

---

## Где хранится словарь

Слова, которые ты добавляешь во вкладке «Мой словарь», сохраняются **локально на устройстве** (в памяти браузера). На телефоне и на ПК это два независимых словаря — они не синхронизируются между собой.

Для переноса или бэкапа пользуйся кнопкой **Экспорт (CSV)** — она выгрузит весь словарь файлом, который открывается в Excel.

---

Готово. Если что-то не загрузилось — чаще всего причина в том, что папка `icons` залилась не полностью или Pages ещё не успел собраться (подожди пару минут).
