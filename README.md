# Latviešu treneris

Тренажёр латышского: глагол *būt* по временам, маркеры времени, склонения существительных, словарь уроков и личный словарь. Работает офлайн, ставится как приложение на iPhone и на Windows.

---

## Файлы

```
index.html            ← «движок» приложения (меняешь редко)
data.json             ← ВСЁ учебное содержание: таблицы, правила, слова
manifest.json         ← описание приложения (имя, иконка)
service-worker.js     ← офлайн-работа и установка
icons/                ← иконки
README.md             ← этот файл
```

Главное новшество: всё учебное содержание лежит в **data.json**. Чтобы добавить слова или правила, ты меняешь только этот файл — index.html трогать не нужно.

## Первая публикация (один раз)

### Шаг 1. Репозиторий на GitHub
1. https://github.com → **New repository**.
2. Имя: `latviesu-treneris`, поставь **Public**, нажми **Create repository**.

### Шаг 2. Залей файлы
1. **Add file → Upload files**.
2. Перетащи все файлы. Папку `icons` заливай так: на странице загрузки впиши в поле имени `icons/` и перетащи туда сами картинки (5 png). Либо залей корневые файлы, потом отдельным заходом — содержимое icons.
3. **Commit changes**.

### Шаг 3. Включи GitHub Pages
1. **Settings → Pages**.
2. *Source*: **Deploy from a branch**, ветка **main**, папка **/ (root)**, **Save**.
3. Через 1–2 минуты появится ссылка: `https://ТВОЙ-ЛОГИН.github.io/latviesu-treneris/`.

### Шаг 4. Установка на iPhone
1. Открой ссылку **в Safari**.
2. **Поделиться** (квадрат со стрелкой) → **На экран «Домой»** → **Добавить**.

### Шаг 5. Установка на Windows
1. Открой ссылку в **Chrome** или **Edge**.
2. Значок установки в адресной строке (или меню → **Установить приложение**).

## Как обновлять содержание потом (главное)

Когда после урока нужно добавить слова, правила или примеры:

1. Открой на GitHub файл **data.json** → карандаш (**Edit**).
2. Внеси изменения (или вставь новый файл, который я тебе дам).
3. **Commit changes**.

Всё. **index.html менять не надо.** Приложение при следующем открытии с интернетом само скачает свежий data.json и обновит содержание. Без интернета покажет последнюю загруженную версию.

> Формат data.json строгий: нельзя терять запятые и кавычки. Если правишь сам и где-то ошибёшься — приложение покажет красное сообщение об ошибке вместо пустого экрана. Надёжнее всего присылать мне материалы урока и заливать готовый файл, который я верну.

### Если менялся сам index.html или иконки (редко)
Тогда открой `service-worker.js` и подними версию:
```js
const CACHE = 'lv-treneris-v3';   // → v4, v5 …
```
Это заставит уже установленные приложения подтянуть новую оболочку. Для обновления **только data.json** менять версию НЕ нужно.

## Обновление того, что уже стоит на устройстве

Раз приложение у тебя уже установлено со старой версией:
1. Перезалей на GitHub все файлы из этого архива (поверх старых), включая новый `data.json` и `service-worker.js` (в нём версия уже поднята до v3).
2. На телефоне/ПК закрой приложение полностью и открой заново — оно обновит оболочку и подхватит data.json.
3. Если содержание не сменилось сразу — открой ещё раз (service worker обновляется со второго запуска).

## Словарь и данные

- **Словарь уроков** (готовые слова) — приходит из data.json, обновляется через веб.
- **Мой словарь** (твои слова) — хранится локально на устройстве, не трогается при обновлениях. Для бэкапа — кнопка **Экспорт (CSV)**.
